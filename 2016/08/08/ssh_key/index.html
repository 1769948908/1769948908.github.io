mac下配置多个ssh_key

发表于 2016-08-08   |   分类于 git

背景

我们经常在同一台电脑向不同的仓库提交代码，比如一个是是要使用公司的git仓库，一个是自己的github，为了不用每次都输入密码，我们会配置ssh key。

在mac下配置多个ssh_key

首先打开终端输入：

ssh-keygen -t rsa -C "email@qq.com"
输入文件名称：如id_rsa_githu_1
Enter file in which to save the key (/Users/huifei.lhf/.ssh/id_rsa): id_rsa_githu_1 
一路Enter下去  

创建第二个
ssh-keygen -t rsa -C "email@qq.com"
输入文件名称：如id_rsa_githu_1
Enter file in which to save the key (/Users/huifei.lhf/.ssh/id_rsa): id_rsa_githu_2
一路Enter下去

按照上面的步骤创建私钥和公钥

添加代理

查看系统ssh-key代理,执行如下命令

  huifeilhfdeMacBook-Air:Documents huifei.lhf$ ssh-add -l
The agent has no identities.
huifeilhfdeMacBook-Air:Documents huifei.lhf$ exec ssh-agent bash
bash-3.2$

如果已有代理，可以删除

$ ssh-add -D

把 .ssh 目录下的2个私钥添加到 ssh-agent 

$ ssh-add ~/.ssh/id_rsa_github_1
$ ssh-add ~/.ssh/id_rsa_github_2

在.ssh下创建 config文件

vi config

在config文件下添加一下内容

Host github.com  
        HostName github.com  
    PreferredAuthentications publickey
    IdentityFile ~/.ssh/id_rsa_github_1

Host github.com 
        HostName github.com  
    PreferredAuthentications publickey
    IdentityFile ~/.ssh/id_rsa_github_2

至此可以使用多个ssh_key了
